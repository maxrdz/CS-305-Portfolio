# Artemis Financial Security Portfolio

This repository contains my Artemis Financial Vulnerability
Assessment Report for CS 305. Artemis Financial needed a
secure software application to protect client financial data.
I identified vulnerabilities, implemented mitigations, and
ensured secure, functional code. Key challenges included
encryption and input validation, and I used testing and
scanning tools to verify security after refactoring. This
project shows my ability to assess risks, apply secure coding
practices, and produce work I can share with future employers.
